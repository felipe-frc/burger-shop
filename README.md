# 🍔 Burger Shop

Projeto de interface web para uma hamburgueria, desenvolvido com foco em **design moderno, responsividade e interatividade com JavaScript**.

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

* Construção de interfaces com HTML e CSS
* Uso do Tailwind CSS
* Manipulação do DOM com JavaScript
* Organização de arquivos front-end

---

## 🚀 Funcionalidades

* Exibição de cardápio
* Interface moderna e responsiva
* Interações com JavaScript
* Layout adaptado para diferentes dispositivos

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* Tailwind CSS
* JavaScript

---

## 📁 Estrutura do projeto

```text
assets/
styles/
index.html
script.js
package.json
```

---

## ▶️ Como executar o projeto

1. Abra o arquivo `index.html` no navegador

ou

2. Utilize uma extensão como Live Server no VS Code
